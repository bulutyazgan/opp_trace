# Opp Trace

Functionality Flow:

The Luma website which is used to register for hackathons has participient lists on their hackathon listings.

The agent will be given a link for an hackathon.

The agent will scrape the website for a guest list, gather everyone

Profiles on luma have information about how many hackathons they have attended in the past.

Almost all luma profiles also have linkedin handles attached to them.

Go to their linkedin profiles, scrape their profiles.

Analyze the profile with AI, check past/recent activity, check how many hackathons they won, got a good place in, in the past. Rate their experience in CS out of a score of 100.

Do this to every attender in the event, list them and rank them according to their 1 - hackathon wins, 2 - experience scores.
